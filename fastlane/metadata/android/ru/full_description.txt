<b style="color:dodgerblue;">ColorBlendr</b> это приложение для изменения цветов Material You дизайна вашего устройства.
Расширьте свой творческий потенциал с помощью простой настройки материала. Мгновенно настраивайте цвета для персонализации всего за несколько нажатий.

<b>Особенности:</b>

- Изменение акцентного насыщения
- Изменение фонового насыщенная
- Изменение фоновой яркости
- Кромешно черная тема в тёмном режиме
- Ручное переопределение цвета
- И многое другое...

<b>Требования:</b>

- Устройство Android 12+
- Доступ root или Shizuku environment

<b>Инструкции:</b>

- Установите и откройте приложение
- Дайте разрешения для приложения
- Выберите root или режим Shizuku
- Вот и все. Теперь вы можете настраивать!

<b>Перевод:</b>

Помогите в переводе ColorBlendr на ваш предпочитаемый язык через <a href="https://crowdin.com/project/colorblendr">нашей платформе Crowdin</a>. Ваш вклад поможет сделать ColorBlendr доступным для более широкой аудитории.

<b>Для чего разрешения:</b>

- <u>QUERY_ALL_PACKAGES </u>: Требуется для индивидуальной темизации сторонних приложений которые имеют поддержку монетной темы но все еще не придерживаются системным цветам.
- <u>MANAGE_EXTERNAL_STORAGE</u>: Требуется для захвата текущих обоев и извлечения доминирующих цветов из обоев.
